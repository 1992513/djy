<tr><td VALIGN=TOP width="626"><h2 align=center>法国里约奥运获奖牌数破战后纪录</h2>
<img width="600" src="https://i.epochtimes.com/assets/uploads/2016/08/GettyImages-593238616-600x400.jpg" />
<h6>图为男子拳击手Tony Yoka（左二）为法国摘第10枚金牌。（Alex Livesey/Getty Images）
</h6>
<hr>
	<p>【大纪元2016年08月22日讯】（大纪元记者张妮法国报导）法国队在2016年里约奥运会上，获得奖牌数破纪录了。8月21日闭幕式前夕，法国男子拳击手Tony Yoka喜获金牌，为法国金牌数增加至10枚，总奖牌数42枚（其中银牌18枚，铜牌14枚），排名第七，破8年前北京奥运的纪录。</p>
<p>这是法国体育代表团在奥运会创下的新纪录，也是法国队在战后创下的纪录。</p>
<p>如果追溯到1900年巴黎奥运会，法国当时获得奖牌91枚，其中金牌27枚。不过，当时的奥运会主要局限在欧美国家，参加国总共24国，其中17个是欧洲国家。</p>
<div style="padding: 0; margin: 0; text-align: left;"><a style="color: #a7a7a7; text-decoration: none; font-weight: normal !important; border: none; display: inline-block;" href="http://www.gettyimages.com/detail/593239144" target="_blank">Embed from Getty Images</a></div>
<div style="overflow: hidden; position: relative; height: 0; padding: 66.666667% 0 0 0; width: 100%;"><a style="display: inline-block; position: absolute; top: 0; left: 0; width: 100%; height: 100%; margin: 0;" src="//embed.gettyimages.com/embed?assets=593239144,593239134,593235084,593236846,593236850&amp;et=pyS1JuwJSypSpvkl1rpRDg&amp;viewMoreLink=on&amp;sig=3ZioFK6K-zFHlrc3HJzhvIZZAiERxNVdJ9j26hUyBZw=&amp;caption=true" width="594" b="396" frameborder="0" scrolling="no"></a></div>
<p style="margin: 0;">
<p>今年法国队出征里约，阵容也是有史以来规模最大的，运动员人数共计396名，其中228名男子和168名女子，代表团的阵容超过了1992年巴塞罗那奥运的335人。</p>
<p>在获金牌的项目上分别有：马术、皮划艇、双人赛艇、男、女子柔道、帆板、男、女子拳击等。在奥运开幕前，法国方面一度曾估计可拿11枚金牌。<br />
法国队这次虽然成绩喜人，但也有多名冠军级名将在赛场上因种种原因受挫，而失去冠军的宝座。</p>
<div class="getty embed image" style="background-color: #fff; display: inline-block; font-family: 'Helvetica Neue',Helvetica,Arial,sans-serif; color: #a7a7a7; font-size: 11px; width: 100%; max-width: 594px;">

<div style="overflow: hidden; position: relative; height: 0; padding: 67.845118% 0 0 0; width: 100%;"><a style="display: inline-block; position: absolute; top: 0; left: 0; width: 100%; height: 100%; margin: 0;" src="//embed.gettyimages.com/embed?assets=593239744,593239732,593240506,593238434,593241204&amp;et=cwzvm2rZQKlroAqSyeOyNA&amp;viewMoreLink=on&amp;sig=xUIoHBloIzZeK-YHuuMI2tRPnmpnIPork96N9y5Jb5Q=&amp;caption=true" width="594" b="403" frameborder="0" scrolling="no"></a></div>

</div>
<p>明显的例子有法国撑杆跳高巨星雷诺．拉维勒尼（Renaud Lavillenie），他曾于2014年以6.16米成绩打破世界纪录，这次里约奥运比赛中，他最后仅以5.98米获银牌。其次是法国男子竞走名将Yohann Diniz本可望摘得这次奥运金牌，万万没想到在19日比赛的当天，起程遥遥领先的他，身体突然发生不适，腹泻让他最后以第8名走完全程。</p>
<p>另外两名游泳名将Florent Manaudou和Camille Lacourt也在赛场失利，失去拿金牌的机会。</p>
