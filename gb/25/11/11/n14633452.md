<tr><td VALIGN=TOP width="626"><h2 align=center>调查：昆州偏远及非都市学校学生安全感最低</h2>
<img width="600" src="https://i.epochtimes.com/assets/uploads/2023/08/id14065115-GettyImages-2462874-600x400.jpg" />
<h6>一项针对昆州1000多所公立学校学生个人调查的分析发现，在安全感评分最低的学校中，非都市地区和偏远地区的学校占大多数。图为示意图，与本文无关。 (Scott Barbour/Getty Images)</h6>
<hr>
<p>【大纪元2025年11月11日讯】（大纪元记者夏楚君澳洲悉尼编译报导）一项针对昆州1000多所公立学校学生个人调查的分析发现，在安全感评分最低的学校中，非都市地区和偏远地区的学校占大多数。</p>
<p>根据《信使邮报》的调查，2024年，有128所学校至少有一名学生回答“在学校感到不安全”，其中77所位于非大都市地区或偏远地区，都市地区则有51所。</p>
<p>在Agnes Water State School州立学校，仅有四分之一的学生表示感到安全，与2023年的89%相比，安全感大幅下降。</p>
<p>Thargomindah State School州立学校学生的安全感更是从2022年的100%骤降至2024年的40%。</p>
<p>其他学生安全感低于50%的学校包括Benaraby State School、Calliope State High School、Mareeba State High School、Charters Towers State High School、Emerald State High School、Northern Beaches State High School和Townsville State High School。</p>
<p>在东南地区，表现最差的学校包括Dakabin State High School、Brisbane Central State School、Caboolture State High School、Bremer State High School和Laidley State High School，这些学校中感到安全的学生比例均低于60%。</p>
<p>另一方面，Cunnamulla P-12 State School的安全感改善幅度最大，从2022年的54%提升至2024年的100%。</p>
<h2>网络欺凌率飙升80%</h2>
<p>校园内安全感下降的同时，昆州儿童的网络欺凌投诉在四年间激增近80%。</p>
<p>eSafety委员会在2022至2025年间共收到昆州儿童网络欺凌投诉1705宗。报告数量从2022年的284宗增加至2024年的515宗，截至今年10月31日已有504宗投诉。</p>
<p>该委员会发言人表示，超过一半的网络欺凌投诉涉及12至15岁的学生，其中Instagram是最常见的平台，其次为TikTok和Snapchat。</p>
<p>儿童求助热线（Kids Helpline）虚拟服务经理托尼·菲茨杰拉德（Tony FitzGerald）表示，2024年昆州儿童的求助电话占全国总数的20%。</p>
<p>他说：“这表明这个问题（网络欺凌）在青少年的生活中仍然普遍存在。”</p>
