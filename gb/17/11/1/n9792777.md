<tr><td VALIGN=TOP width="626"><h2 align=center>乐舞文学赏析：楚辞．九歌．东君</h2>
<img width="600" src="https://i.epochtimes.com/assets/uploads/2017/11/1711010158292357-600x400.jpg" />
<h6>元 张渥九歌图──东君。（公有领域）
</h6>
<hr>
<p style="text-align: center;"><strong><span style="font-family: 标楷体; font-size: medium; color: #993300;">《<ahref="https://github.com/19920513/djy/blob/master/gb/tag/%E6%A5%9A%E8%BE%9E.md#1">楚辞</a>．九歌．<ahref="https://github.com/19920513/djy/blob/master/gb/tag/%E4%B8%9C%E5%90%9B.md#1">东君</a>》</span></strong></p>
<p style="text-align: center;"><strong><span style="font-family: 标楷体; font-size: medium; color: #993300;">暾将出兮东方，照吾槛兮扶桑。</span>[1]</strong><br />
<strong> <span style="font-family: 标楷体; font-size: medium; color: #993300;">抚余马兮安驱，夜皎皎兮既明。</span>[2]</strong><br />
<strong> <span style="font-family: 标楷体; font-size: medium; color: #993300;">驾龙辀兮乘雷，载云旗兮委蛇。</span>[3]</strong><br />
<strong> <span style="font-family: 标楷体; font-size: medium; color: #993300;">长太息兮将上，心低徊兮顾怀。</span>[4]</strong><br />
<strong> <span style="font-family: 标楷体; font-size: medium; color: #993300;">羌声色兮娱人，观者儋兮忘归。</span>[5]</strong><br />
<strong> <span style="font-family: 标楷体; font-size: medium; color: #993300;">緪瑟兮交鼓，萧钟兮瑶簴，</span>[6]</strong><br />
<strong> <span style="font-family: 标楷体; font-size: medium; color: #993300;">鸣篪兮吹竽，思灵保兮贤姱。</span>[7]</strong><br />
<strong> <span style="font-family: 标楷体; font-size: medium; color: #993300;">翾飞兮翠曾，展诗兮会舞。</span>[8]</strong><br />
<strong> <span style="font-family: 标楷体; font-size: medium; color: #993300;">应律兮合节，灵之来兮蔽日。</span>[9]</strong><br />
<strong> <span style="font-family: 标楷体; font-size: medium; color: #993300;">青云衣兮白霓裳，举长矢兮射天狼。</span>[10]</strong><br />
<strong> <span style="font-family: 标楷体; font-size: medium; color: #993300;">操余弧兮反沦降，援北斗兮酌桂浆。</span>[11]</strong><br />
<strong> <span style="font-family: 标楷体; font-size: medium; color: #993300;">撰余辔兮高驼翔，杳冥冥兮以东行。</span>[12]</strong></p>
<p><strong>参考注释：</strong></p>
<p><strong>[1]</strong>暾（tūn）：意指初升之太阳或形容其光芒炽盛。槛：车上之围栏、栏杆。扶桑：传说中的神树，也可指称东方极远处或太阳出来的地方。《文选．日出东南隅行》：<span style="color: #000080;">扶桑开朝晕，此高台端。</span></p>
<p><strong>[2]</strong>余：我，日神自称。安驱：缓步徐行。皎皎：明亮貌。</p>
<p><strong>[3]</strong>辀（zhōu）：车上之横木，也泛指车。龙辀：以龙为车，雷：指以雷为车轮，所以说是乘雷。云旗：用云当作自己的旗帜。王逸注：“<span style="color: #000080;">以云为旌旗。</span>” 委蛇：可解释为从容自得的样子或蜿蜒曲折意思。</p>
<p><strong>[4]</strong>太息：大声长叹，深深地叹息的意思，低徊：迟疑不进。顾怀：眷恋、回顾的意思。</p>
<p><strong>[5]</strong>羌声色: 羌为发语词，声色：美好的声音与颜色（应指下一段的乐舞）。娱人：使人欢乐，憺（dàn）：安逸的意思。</p>
<p><strong>[6]</strong>緪（gēng）：指急促地弹奏，交鼓：指鼓声应和瑟之音，萧钟：用力撞钟，瑶：通摇，震动的意思。簴（jù）：悬乐器的架子。瑶簴指因钟响而让簴也产生共鸣。</p>
<p><strong>[7]</strong>篪（chí）：古代的管乐器，用竹制成。灵保：指巫师。贤：美德。姱（hǔ）：美貌。</p>
<p><strong>[8]</strong>翾飞：翾（xuán），指轻柔地飞扬。翠曾：飞起，形容舞姿。展诗：展开诗歌来唱。会舞：指众巫合舞。</p>
<p><strong>[9]</strong>应律：指协和音律。合节：指舞合节拍。灵：神灵。</p>
<p><strong>[10]</strong>青云衣兮白霓裳：指日神之服装。矢：箭。天狼：即天狼星，古以为主侵掠。王逸注：“<span style="color: #000080;">天狼，星名，以喻贪残。</span>”苏轼〈江城子〉:“<span style="color: #000080;">会挽雕弓似满月，西北望，射天狼</span>”，他以天狼星比拟威胁北宋边境的西夏。也有学者以为此篇天狼是比喻西方的秦国。</p>
<p><strong>[11]</strong>弧：木制的弓。反：指返身西向。沦降：指太阳西沉。援：引。北斗：星名，由七颗星组成，形似舀汤的勺。桂浆：桂花酿的酒。</p>
<p><strong>[12]</strong>撰：持着。驼翔：驰骋飞翔。王逸 注：“<span style="color: #000080;">驼，一作驰。</span>”洪兴祖 补注：“<span style="color: #000080;">高驰翔者，喻制世驭民于万物之上。</span>”杳：幽深。冥冥：黑暗样子。</p>
<p><strong>参考译文</strong></p>
<p><strong><span style="font-family: 标楷体; font-size: medium; color: #993300;">东方即将升起温和而明亮的太阳，他将车驾的栏木及扶桑神木照耀得闪闪发光。</span></strong></p>
<p><strong><span style="font-family: 标楷体; font-size: medium; color: #993300;">我手轻拍着马驾车徐行？沉沉的夜色将结束，天色慢慢变亮。</span></strong></p>
<p><strong><span style="font-family: 标楷体; font-size: medium; color: #993300;">我驾着龙车踏着雷电，乘着四周的白云就如同我的旗帜，随风飘扬自在的飞扬(也可解释为高举的云旗正随风飘扬)。</span></strong></p>
<p><strong><span style="font-family: 标楷体; font-size: medium; color: #993300;">在长声叹息中神灵将升空返回天庭，然而心里却是迟疑不决，仍眷怀着下界众生。</span></strong></p>
<p><strong><span style="font-family: 标楷体; font-size: medium; color: #993300;">美好的音乐与舞蹈令人心生眷恋，让四方的众生安逸其中，流连忘返。</span></strong></p>
<p><strong><span style="font-family: 标楷体; font-size: medium; color: #993300;">急促的瑟音与咚咚的鼓声应和着，钟声也在这时响起，让钟架也产生共鸣摇摆不已。</span></strong></p>
<p><strong><span style="font-family: 标楷体; font-size: medium; color: #993300;">乐手吹奏着竽、篪，祭祀的巫师们是德行与美丽兼具。</span></strong></p>
<p><strong><span style="font-family: 标楷体; font-size: medium; color: #993300;">舞师跳着轻柔的舞步翩然飞起，唱和着美妙的诗歌，我们一同群起共舞。</span></strong></p>
<p><strong><span style="font-family: 标楷体; font-size: medium; color: #993300;">应和着歌舞的旋律伴着节奏，纷纷降世而来的神灵遮蔽着阳光。</span></strong></p>
<p><strong><span style="font-family: 标楷体; font-size: medium; color: #993300;">我穿上如青云的上衣，白霓的裙裳，手持着弧弓将长箭射向天狼星。</span></strong></p>
<p><strong><span style="font-family: 标楷体; font-size: medium; color: #993300;">这时太阳西沉了，我收起弧弓返回车上，手拿着北斗星勺喝着桂花泡的美酒。</span></strong></p>
<p><strong><span style="font-family: 标楷体; font-size: medium; color: #993300;">我手牵起缰绳再次高飞翱翔，在幽深的夜色中回到东方。</span></strong></p>
<p><strong>题解及赏析：</strong></p>
<p>《<ahref="https://github.com/19920513/djy/blob/master/gb/tag/%E4%B8%9C%E5%90%9B.md#1">东君</a>》一诗是应是祭祀太阳神的祭祀辞，内容生动地描述了祭典的盛况，然而内容隐约的也可体会到了作为主角的太阳神受到了人们虔诚祭祀的诚心所感动，因而击败天狼星，也是光明战胜黑暗的一段故事。</p>
<p>洪兴祖《<ahref="https://github.com/19920513/djy/blob/master/gb/tag/%E6%A5%9A%E8%BE%9E.md#1">楚辞</a>补注》云：<span style="color: #000080;">《博雅》曰：朱明、耀灵、东君，日也。</span><span style="color: #000080;"><span style="color: #000000;">朱熹《楚辞集注》云：</span>“此日神也。”</span></p>
<p>清朝学者戴震《屈原赋注》：<span style="color: #000080;">此歌备陈歌舞之事，盖举迎日典礼赋之</span>。</p>
<p>诗一开头就描述了太阳神东君的住所，在黎明时刻，温煦明亮的日光开始慢慢照耀大地，似乎也象征着在祭坛四周人民的翘首盼望。</p>
<p>之后太阳神东君从容不迫地驾着他的龙车缓缓前进，雷电声震万里，伴随着四周翻腾如海的白云，为东君的出场增添了不少豪气。</p>
<p>然而此时太阳神看着下界众生表演的乐舞，心中产生了一丝丝的眷恋：<strong>“</strong><strong><span style="color: #000080;">长太息兮将上，心低徊兮顾怀</span>。”</strong></p>
<p>这一段将神与世人的距离拉得更近，让严肃的太阳神多了几许的人情、可亲。接着众生们弹琴，敲着钟鼓，吹奏竽、篪……。乐器共鸣出美妙的音乐，这时具有美德的巫师出场了，他唱和着美妙的诗歌，带领着人们一同群起共舞。</p>
<p>这时的人们载歌载舞“<span style="color: #000080;"><strong>翾飞兮翠曾，展诗兮会舞</strong></span>” 轻柔的舞步翩飞，与交响乐、声乐融合成了一个美丽的画面，反映了人们对太阳神的崇敬、信仰，对追求光明的渴望。</p>
<p>这时东君被祭神的乐舞所感动，带着众神们纷纷下世，将天空都遮蔽了，或许这时的时刻正好是日正当中，太阳神与诸神们只是停留空中，或许是仍在任务中无暇下世，或许是他的无边光芒及热会伤及众生，但他已接收到了众生“为神而舞”的敬意，接受了人民的祭祀后便戏剧性的为民除害，向带来灾祸的天狼星挑战。</p>
<p>《晋书．天文志》：“<span style="color: #000080;">狼一星，在东井东南。狼为野将，主侵掠。</span>”张守节《史记正义》云：“<span style="color: #000080;">弧九星，在狼东南，天之弓也。以伐叛怀远，又主备贼盗之知奸邪者。</span>”</p>
<p>天狼星象征黑暗、灾祸的一方。只见太阳神穿起战服，如青云的上衣、白霓的裙裳，挽弓拉箭朝着天狼星射去，将之化为尘埃。击败天狼星后的太阳神又端起像勺子状的北斗七星斟满了桂花酿的美酒一饮而尽，其英雄本色显露无遗。夜幕低垂后才拉起了缰绳，上车再次高飞翱翔，在幽深的夜色中返回东方。</p>
<p>这首诗篇体现了中国传统文化“天人合一”及“光明战胜邪恶”的深刻内涵，在传统文化中光明战胜黑暗的故事可谓多不胜数，在西方也有着许多类似的故事。</p>
<p>经典的歌剧《魔笛》讲述的就是一例：在中世纪的世界，当时天地陷入失衡的状态，日与夜永远地分开了。在夜的世界里万物无法繁殖。在日的世界中万物得不到休养生息，人类苦不堪言。</p>
<p>这时在东方的国度有一位王子塔米诺，他为了解救人民，放下王族的光环独自一人别离家园，前往夜后的国度找寻解除诅咒的方法。途中他接受了夜后救出其女帕蜜娜公主的委托，同时也得知了破除诅咒之法就是要找回失去的大日轮。</p>
<p>在途中他运用了夜后给的魔笛，吹奏着天籁之音收服一路上的魔物，同时救出了公主，然而塔米诺王子又面临着祭司给的三个考验：接受挑战的“勇气”、“沉默”、及“水”与“火”。最后王子靠着坚定的信念及公主的协助下通过重重考验，脱胎换骨，成了日之男的一分子。故事最终，圣者萨拉斯将大日轮传承给了塔米诺，世界的秩序也因而恢复。</p>
<p>神传文化在东西方虽有着不同的体系，然而对于人的教化有着许多的共同点，观阅经典的诗词、歌剧可体会到人们虔诚向善的心以及对神信仰意志的坚定，〈东君〉这首诗歌也正体现了这一点。@#</p>
