<tr><td VALIGN=TOP width="626"><h2 align=center>捷运首创  桃捷推扫QR Code就打扫服务“立码清”</h2>
<img width="600" src="https://i.epochtimes.com/assets/uploads/2023/12/id14133784-652439-600x400.jpg" />
<h6>捷运系统首创 桃捷推扫QR Code就打扫服务，秒速通报“立码清”。（桃园机场捷运提供）
</h6>
<hr>
<p>【大纪元2023年12月11日讯】（大纪元记者徐乃义台湾桃园报导）桃园机场捷运线上发现垃圾或脏污，人人可以成为护洁达人！拿起手机扫一下“Q洁净立码清服务”的 QR Code（行动条码），桃园大众捷运股份有限公司将即刻收到讯息，即时派员清理，还一个干净的乘车空间给大家。</p>
<p>通常在交通运输业上最被诟病的状况是打翻饮料、呕吐秽物、卫生纸屑、垃圾布满地、厕所地板尿液四贱、马桶粪便池脏污或厕所内垃圾桶卫生纸满溢未清倒等等，让人却步不前。这些状况也让清洁工作人员疲于奔命巡回打扫不休，可是仍难敌破坏环境清洁的状况无止境的不断出现。镇守国门，国际入出境旅客多，桃园机场捷运线乘车空间一直保持的相当干净，也给国际旅客留下良好印象。</p>
<p>为保持这项模范风评，桃园大众捷运股份有限公司利用5G科技推出“Q洁净立码清服务”，建置快速通报清洁反应的管道，12日起在捷运车厢及车站发现垃圾或脏污，拿起自己手机扫一下桃园大众捷运（股）公司制作的“Q洁净立码清服务”的QR Code（行动条码），讯息将立即反馈回桃园大众捷运（股）公司通报管道，当通报管道接收到“紧急清洁服务”的需求后，随即通知清洁人员到场检视清理，再回报清理状况，让捷运车厢或车站空间回复清洁状态。</p>
<p>在桃园机场捷运线的捷运车厢内每一扇车门边都张贴有“Q洁净立码清服务”的QR Code （行动条码），每一节捷运车厢有六扇门，一列捷运有四节车厢，总共就贴了24张“Q洁净立码清服务”QR Code （行动条码），方便大家扫码。</p>
<p>于全线车站的询问处、厕所及厕所内的垃圾桶上，也张贴有“Q洁净立码清服务”的QR Code（行动条码），方便大家随时扫码，随时反应，达到除污务净。</p>
<p>“Q洁净立码清服务”的QR Code（行动条码）操作相当便利，拿起手机扫一下“Q洁净立码清服务”的QR Code （行动条码），照着指示点进去，把发现污秽或脏乱地点标示出来，不用1秒钟时间，讯息就快速传送到桃园大众捷运（股）公司清洁维謢的通报管道， 让桃园机场捷运线从车厢到车站一路维持在干净、清洁、舒适的空间状态，乘客好感度一再提升。◇</p>
