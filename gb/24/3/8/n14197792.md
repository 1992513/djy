<tr><td VALIGN=TOP width="626"><h2 align=center>西澳家长给公立学校自愿交费减少</h2>
<h6></h6>
<hr>
	<p>【大纪元2024年03月08日讯】<span lang="zh-CN"><span lang="zh-TW">（大纪元记者林达澳洲珀斯编译报导）自疫情爆发以来，</span><span lang="zh-TW">生活</span><span lang="zh-TW">成本持续上升，向公立学校自愿交费的西澳家长数量有所下降。</span><span lang="zh-TW">?</span></span></p>
<p><span lang="zh-CN"><span lang="zh-TW">据“</span></span>WA Today<span lang="zh-CN"><span lang="zh-TW">”新闻网报导，</span></span>2019 <span lang="zh-CN"><span lang="zh-TW">年，在</span> </span>COVID-19 <span lang="zh-CN"><span lang="zh-TW">封锁之前，公校向每名学生收取学杂费、家长自愿交费</span> </span>357 <span lang="zh-CN"><span lang="zh-TW">澳元。</span></span>2020 <span lang="zh-CN"><span lang="zh-TW">年，这一数字下降了</span> </span>100 <span lang="zh-CN"><span lang="zh-TW">澳元，虽然此后一直在缓慢攀升，但仍低于</span> </span>2022 <span lang="zh-CN"><span lang="zh-TW">年大流行前的水平，即每位学生</span> </span>294 <span lang="zh-CN"><span lang="zh-TW">澳元。</span><span lang="zh-TW">?</span></span></p>
<p><span lang="zh-CN"><span lang="zh-TW">澳大利亚课程评估报告局（</span></span>ACARA<span lang="zh-CN"><span lang="zh-TW">）日前发布了新数据。传统上，学费为非政府部门创造了大量收入，所有年级的学费都是强制性的。</span><span lang="zh-TW">?</span></span></p>
<p><span lang="zh-CN"><span lang="zh-TW">公校小学自愿捐款最多</span> </span>60 <span lang="zh-CN"><span lang="zh-TW">澳元，</span></span>7 <span lang="zh-CN"><span lang="zh-TW">至</span> </span>10 <span lang="zh-CN"><span lang="zh-TW">年级最多</span> </span>235 <span lang="zh-CN"><span lang="zh-TW">澳元。</span></span>11 <span lang="zh-CN"><span lang="zh-TW">年级和</span> </span>12 <span lang="zh-CN"><span lang="zh-TW">年级费用是唯一强制性的。</span><span lang="zh-TW">?</span></span></p>
<p><span lang="zh-CN"><span lang="zh-TW">总体而言，西澳私校</span> </span>40% <span lang="zh-CN"><span lang="zh-TW">的收入来自学费，而公校的这一比例仅为</span> </span>1.6%<span lang="zh-CN"><span lang="zh-TW">。</span><span lang="zh-TW">?</span></span></p>
<p><span lang="zh-CN"><span lang="zh-TW">基于社会教育优势的公立学校之间的收入差距也很明显。西郊的</span></span>Shenton College <span lang="zh-CN"><span lang="zh-TW">中学学费为</span> </span>727 <span lang="zh-CN"><span lang="zh-TW">澳元，但全校全年收入接近</span> </span>200 <span lang="zh-CN"><span lang="zh-TW">万澳元，这表明富裕地区家长能够为学校做出更大的贡献。该区只有</span> </span>3% <span lang="zh-CN"><span lang="zh-TW">的学生处于社会经济优势下四分位数，而</span> </span>65% <span lang="zh-CN"><span lang="zh-TW">的学生处于社会经济优势上四分位数</span> <span lang="zh-TW">。</span><span lang="zh-TW">?</span></span></p>
<p><span lang="zh-CN"><span lang="zh-TW">但在</span></span>Gilmore College<span lang="zh-CN"><span lang="zh-TW">高中，</span></span>65% <span lang="zh-CN"><span lang="zh-TW">的学生处于社会经济优势下四分位数，学生平均交费为</span></span>206 <span lang="zh-CN"><span lang="zh-TW">澳元，即全年略高于</span> </span>24 <span lang="zh-CN"><span lang="zh-TW">万澳元。</span><span lang="zh-TW">?</span></span></p>
<p><span lang="zh-CN"><span lang="zh-TW">然而，与大流行之前相比，即使是</span></span>Shenton College<span lang="zh-CN"><span lang="zh-TW">的家长自愿交费也有所减少</span><span lang="zh-TW">——</span></span>2019 <span lang="zh-CN"><span lang="zh-TW">年，学生平均交费</span></span>956 <span lang="zh-CN"><span lang="zh-TW">澳元。</span><span lang="zh-TW">?</span></span></p>
<p><span lang="zh-CN"><span lang="zh-TW">尽管家长支出减少，但州政府的资助从</span> </span>2018-2019 <span lang="zh-CN"><span lang="zh-TW">财年的每名学生</span> </span>16,652 <span lang="zh-CN"><span lang="zh-TW">澳元增加到</span> </span>2021-2022 <span lang="zh-CN"><span lang="zh-TW">年每名学生</span> </span>18,435 <span lang="zh-CN"><span lang="zh-TW">澳元。</span></span></p>
<p><span lang="zh-CN"><span lang="zh-TW">州和联邦政府最近的一项公告还意味着公校将获得额外</span> </span>16 <span lang="zh-CN"><span lang="zh-TW">亿澳元的资金，以达到学校资源标准</span><span lang="zh-TW">——</span><span lang="zh-TW">学校满足教育所需的最低资金数额评估。</span></span></p>
<p><span lang="zh-TW">责任编辑：杨新云</span></p>
