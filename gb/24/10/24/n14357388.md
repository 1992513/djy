<tr><td VALIGN=TOP width="626"><h2 align=center>国税局鼓励为2025年报税季注册IP PIN</h2>
<h6></h6>
<hr>
	<p>【大纪元2024年10月25日讯】华府 — 2025 年报税季临近，美国国税局鼓励所有纳税人采取重要措施，透过注册身份保护个人识别号码(IP PIN) 来保护自己的身份。</p>
<p>这个简单但关键的步骤可以提供额外一层安全，有助于防止与税务相关的身份盗窃。</p>
<p>国税局鼓励纳税人注册国税局线上账户，该账户提供了一种快速、简单的方法来获取 IP PIN。尽早注册将确保纳税人拥有额外的安全性，因为在 2025 年报税季开始时，纳税人可以使用 IP PIN 以电子方式提交税表。</p>
<p>国税局鼓励纳税人在2024 年 11 月 23 日之前注册 IP PIN。在此日期之后，IP PIN 系统将进行维护，直到2025 年 1 月初才可再使用。现在注册 IP PIN 将确保纳税人的身份在报税季开始时受到保护。在此期间，我们将为 2025 年报税季产生新的 IP PIN，因此线上注册者必须从 2025 年 1 月初开始查询以取得其新的IP PIN。</p>
<p>IP PIN 是一个六位数字，可防止其他人使用纳税人的社会安全号码或个人报税识别号码提交联邦税表。它是确保纳税人个人和财务资讯安全的重要工具。 IP PIN 只有个人和国税局知道，当他们以电子方式提交税表时，IP PIN会确认他们的身份，使窃贼很难使用纳税人的资讯进行诈欺。</p>
<p>如何申请IP PIN</p>
<p>注册 IP PIN 的最佳方式是透过国税局线上账户。这个过程需要身份验证，配偶和被抚养人如果完成所需的验证书步骤，也可以获得 IP PIN。 一旦发放IP PIN，电子和纸本税表上都必须使用它。</p>
<p>纳税人要获得 IP PIN，应在 IRS.gov 建立或登入其线上账户，并按照身份验证步骤进行操作。验证后，纳税人需要点击个人资料标签来申请其 IP PIN。 IP PIN 用户在提交当前日历年以及同期提交的任何往年的联邦税表时必须使用此号码。</p>
<p>纳税人如果无法建立线上账户，可以使用其它方法，例如在纳税人协助中心进行亲自身份验证。有关如何注册的更多信息，请访问获取身份保护个人识别号码 (IP PIN)。</p>
<p>有关 IP PIN 的其它信息</p>
<p>IP PIN 的有效期限为一个日历年。出于安全原因，每日历年初就会产生新的 IP PIN。一些参与者将透过邮件收到其 IP PIN，而其他参与者则必须登入其线上账户才能查看其当前的 IP PIN。</p>
<p>已登记的纳税人可以重新登入其线上账户以查看其目前的IP PIN。拥有IP PIN 码的纳税人在当年提交任何联邦税表（包括上一年税表或修订税表）时必须使用它。IP PIN 使用者应仅与国税局及其报税服务提供者分享其号码。国税局绝不会透过电话、电子邮件或短信件询问 IP PIN。</p>
<p>纳税人现在可为2024年取得IP PIN。国税局将从 2025 年 1 月开始发放新的 IP PIN。加入IP PIN 专案的纳税人，如果没有成为税务相关身份盗窃的受害者，可以透过其线上账户选择退出IP PIN 项目。</p>
